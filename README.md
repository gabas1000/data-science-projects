# Machine Learning Projects

This repository contains selected machine learning and AI projects I completed through coursework and independent technical work. Each folder includes a project-specific README and supporting materials.

## Projects

### [CNN Depth and Hyperparameter Effects on CIFAR-10](./cnn-depth-hyperparameter-study-cifar10)
Experimental study on how CNN depth, optimizer choice, learning rate, pooling, and training duration affect image classification performance on CIFAR-10.

### [Classifier Performance Across Dataset Difficulty](./classifier-performance-across-dataset-difficulty)
Comparative analysis of SVM, Random Forest, and KNN across binary classification datasets with different levels of difficulty and multiple train/test splits.

### [Prompt Phrasing and String Reversal in LLMs](./prompt-phrasing-string-reversal-llms)
Study of how prompt wording affects GPT-3.5 Turbo performance on a symbolic string-reversal task, including error analysis and prompt-level accuracy comparisons.
